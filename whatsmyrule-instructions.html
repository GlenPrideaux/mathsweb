What's My Rule: Instructions

  1. Select difficulty level to make a new rule.
  2. Enter an input number and press 'Calculate' to see what output number the rule gives. Keep doing this until you think you know what the rule is.
  3. Enter a new input number and the output number you think the rule will generate, then press 'Check'. You must do this twice in a row to win.
  4. On the iPad and iPhone versions, shake the device to give up and see what the rule is.

Note that a rule can be written in different ways. You might have written the rule differently than the computer, but still be right.
