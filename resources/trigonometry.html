Introduction to Trigonometric Projections

These web pages provide a series of interactive lessons to introduce the concept of trigonometric functions as projections based on the unit circle. They extend the concept of trigonometric ratios first encountered in right triangles. They are not intended to be used instead of a text book, but as supplementary or extension material to give students another way of exploring the subject material.

Note: These pages are based on the JSP applet from Key Curriculum Press. They require a web browser that correctly supports Java applets. Note that some Java Virtual Machines are not standards-compliant and may fail to run the JSP applet. If this occurs, I recommend obtaining a replacement JVM directly from the Sun Java site.
